# Sensor Readings

This application displays a data set of sensor readings in tabular and graph form, with the ability to sort by time and data type, and filter the graph view for each specific sensor over time (selectable from the table).

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.15.
## Setup

Clone the project using `git clone` and then run `npm i` in the root directory to install
## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
